Инструкции по использованию платформы электронного документооборота

На главной странице приведены следующие вкладки:


1. Организации - содержит список всех добавленных в систему организаций. Перейдя по "Подробнее об организации" происходит переход на страницу данной организации. В ней содержится вся информация о ней, а так же имеются кнопки для удаления организации, редактирования её данных, и кнопка "Подробробнее об подразделениях", которая содержит список подразделений, относящихся к данной организации. Кроме того, при переходе по кнопке "Организации" с главной страницы, в верхней панели появляется кнопка для добавления новой организации.


2. Подразделения - содержит список всех добавленных в систему подразделений. Перейдя по "Подробнее о подразделении" происходит переход на страницу данного подразделения. В ней содержится вся информация о ней, а так же имеются кнопки для удаления подразделения, редактирования её данных, и кнопка "Подробробнее о сотрудниках", которая содержит список всех сотрудников данного подразделения. Кроме того, при переходе по кнопке "Подразделения" с главной страницы, в верхней панели появляется кнопка для добавления нового подразделения.


3. Сотрудники - содержит список всех добавленных в систему сотрудников. Перейдя по "Подробнее о сотруднике" происходит переход на страницу данного сотрудника. В ней содержится вся информация о нём, а так же имеются кнопки для удаления сотрудника, редактирования его данных, и кнопка "Подробробнее о поручениях", которая содержит список всех поручений, активных для данного сотрудника. Кроме того, при переходе по кнопке "Сотрудники" с главной страницы, в верхней панели появляется кнопка для добавления нового сотрудника.


4. Поручения - содержит список всех добавленных в систему порученений. Перейдя по "Подробнее о поручении" происходит переход на страницу данного поручения. В ней содержится вся информация о нём, а так же имеются кнопки для удаления и редактирования поручения.


5. Личный кабинет - перейдя по данной кнопке происходит переход на страницу, содержащую информацию об авторизованном пользователе. Т.е у каждого пользователя данная страница своя. Перейдя по "подробнее о поручениях", происходит переход на страницу, содержащей список поручений, относящихся к авторизованному пользователю. Но в отличие от общего списка поручений, поручения на данной странице подлежат управлению. Т.е в зависимости от того, на какой стадии находится процесс работы с поручением, с помощью кнопки с соответствующим названием можно производить управлением документом(поручением). Правила: Управление поручением должно начинаться с первой стадии, и идти последовательно друг за другом, за исклчючением случая отрицательного прохождения стадии контроля.
